About this Site

This site was built using Jekyll, a static-site generator and is hosted via GitHub Pages. The development enviornment I used to build is is comprised of the following:

  • Jade (a HTML pre-processor)
  • SASS (a CSS pre-processor)
  • Gulp (a build system)
  • Atom (a great text-editor)
